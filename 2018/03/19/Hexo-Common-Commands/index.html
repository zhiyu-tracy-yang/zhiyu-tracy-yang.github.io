Hexo and Theme

Posted on 2018-03-19 Edited on 2020-06-27

Hexo and theme changes.

Create a new post 创建一个新的文章

$ hexo new "new post"

View site static 发布前预览

$ hexo s

Clean cache file 清理缓存（如果修改了格式）

$ hexo clean

Generate and deploy files 发布

hexo g -d

Change header image

themes/next/source/css/_variables/base.styl

Change background image

themes/next/source/css/_common/scaffolding/base.styl, body:background-image
