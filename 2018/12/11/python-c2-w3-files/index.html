File operations in Python 3

Open File

handle = open(fileName, mode)

fhandle = open(‘mbox.txt‘, ‘r‘)

1. Returns a file handle to manipulate the file.
2. Filename is a string.
3. Model is optional. Read is ‘r’ and write is ‘w’.
