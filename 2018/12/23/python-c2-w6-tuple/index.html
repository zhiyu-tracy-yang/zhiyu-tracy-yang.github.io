Data Structure - Tuple in Python 3.

Tuple

Tuple - 元组，或者看成方程组
Like lists, tuples have elements, indexing from 0.
Declare a tuple with “()”, and put elements in it.
Tupels cannot be changed.

Tuple Assignment

We could put a tuple on the left-hand side of assignment.

Tuples are Comparable

If the first item is equal, Python goes to the next element, until it finds elements that differ.

Sorting Lists of Tuples

By Key

The dict.items() returns a list of tuples. We can sort this list using sorted(dict.items()) by key.

By Value

First copy the dictionary to another list, but put keys as values and values as keys. Then sort this list.

List Comprehension

List comprehension creates a dynamic list.
